# Loan Default Prediction 💳

##  Project Overview
This project predicts whether a customer will default on a loan using Machine Learning techniques. It helps financial institutions identify high-risk customers and make better lending decisions.

##  Technologies Used
 Python
 Pandas
 NumPy
 Scikit-learn
 Matplotlib / Seaborn
 Google Colab

##  Dataset
The dataset contains customer-related information such as income, loan amount, credit history, and other financial attributes used to predict loan default.

##  Project Workflow
1. Data Collection
2. Data Preprocessing
    Handling missing values
    Encoding categorical variables
    Feature scaling
3. Exploratory Data Analysis (EDA)
4. Model Building
    Logistic Regression
    Decision Tree
5. Model Evaluation
    Accuracy
    Precision
    Recall

##  Results
The model successfully predicts loan default risk and helps in identifying potential defaulters.
